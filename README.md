## Run for the first time:

### First Step:
```bash
ddev composer
```
### Second Step:
```bash
ddev import-db --file=database/db.sql.gz
```
### Third Step:
```bash
ddev start
```

## After that, perform the following when you run it in the future:
```bash
ddev launch
```
